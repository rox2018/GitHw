Common attributes: src is a mandatory attribute,it is the image URL width is the intrinsic width of the image in pixels. link href contains a URL or a URL fragment that the hyperlink points to. type specifies the media type in the form of a MIME type for the linked URL. It is purely advisory, with no built-in functionality.
autocomplete indicates whether input elements can by default have their values automatically completed by the browser. It can be overridden by an autocomplete attribute on an element belonging to the form. Can be set to on or off. text sets the text content of the element. async is a Boolean attribute indicating that the browser should, if possible, execute the script asynchronously. tel is a control for entering a telephone number button is a push button with no default behavior.
